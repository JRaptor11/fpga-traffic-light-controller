# FPGA Traffic Light Controller

This project implements a traffic light control system on an FPGA using VHDL. The system uses a finite state machine (FSM) to control traffic flow based on timing and sensor inputs.

## Overview

The design models a real-world traffic intersection and includes multiple traffic states, transitions, and safety constraints.

## Key Features

- 14-state finite state machine
- Sensor-based adaptive timing
- Hardware implementation on FPGA (ZedBoard)
- Real-time system behavior
